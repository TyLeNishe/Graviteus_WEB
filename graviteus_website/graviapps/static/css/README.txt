Все стили проекта распределены по файлам, каждый файл отвечает за определённую задачу.  В некоторых файлах параметры стилей вынесены в :root, который расположен начале файла или в base.css
Для большего понаминия в некоторых файлах (почти во всех) присутствуют комментарии.

В файле base указаны базовые переменные для всего сайта (пока указаны не все, но некоторые), шрифт и все базовые настройки.
В папке components расположены компоненты страниц, которые используются на разных страницах (например кнопки, шапка...). 
В папке pages расположены стили для определённых страниц, которые больше нигде не используются (например hero-секция или карусель изображений с главной страницы).
В папке fun расположены стили, отвечающие за интерактивность сайта, но не являющиеся обязательными (например эффект при клике).
Далее представлены предназначения каждого из файлов

==> base.css:
Тут расположены базовые стили и переменные проекта:
- Шрифты
- Базовые стили
- Переменные:
- - Цвета
- - Размеры кнопок (базовые) 
- - Отступы

==> buttons.css:
Тут расположены стили для кнопок (все кнопки, включая ссылки)

==> forms.css:
Тут расположены стили для текстбоксов (полей ввода)

==> cards.css:
Тут расположены стили для карточек (полупрозрачные боксы с элементами, например на страницах входа/регистрации)

==> header.css:
Тут расположены стили для шапки и подвала
==> auth.css:
Тут расположены стили для страниц входа/регистрации (разделители, формы входа)

==> home.css:
Тут расположены стили для главной страницы (карусели, hero-секции)

==> profile.css:
Тут расположены стили для страницы профиля (аккордеоны (сворачиваемые элементы), иконки достижений)

==> shop.css:
Тут будут расположены стили для магазина (пока не используется)


Кнопки:
Есть 3 размера кнопок: маленькие, средние (используются по умолчанию) и большие. Для изменения размера нужно в стиле добавить/изменить стиль btn-*, где * может быть sm, md, lg. (Например style="btn btn-md")
